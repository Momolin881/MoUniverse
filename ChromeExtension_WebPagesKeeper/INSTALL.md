# 🚀 簡易安裝指南（無需圖示）

## ✅ 立即安裝（只需 2 分鐘）

我已經幫你調整好設定，**不需要圖示也能使用**！Chrome 會自動顯示預設圖示。

### 步驟 1：開啟 Chrome 擴充功能頁面

在 Chrome 瀏覽器的網址列輸入：
```
chrome://extensions/
```
然後按 Enter

### 步驟 2：開啟開發人員模式

在右上角找到「**開發人員模式**」開關，點擊開啟（會變成藍色）

### 步驟 3：載入擴充套件

1. 點擊左上角的「**載入未封裝項目**」按鈕
2. 選擇這個資料夾：`ChromeExtension_WebPagesKeeper`
3. 點擊「選擇」

### 步驟 4：完成！

你會看到：
- ✅ 擴充套件出現在列表中
- ✅ 名稱：「WebPages Keeper - Notion助手」
- ✅ 右上角工具列出現拼圖圖示

---

## 🎯 開始使用

### 第一次測試

1. **開啟任何網頁**（例如：https://www.google.com）

2. **點擊擴充套件**
   - 點擊瀏覽器右上角的「拼圖」圖示 🧩
   - 找到「WebPages Keeper - Notion助手」
   - 點擊它

3. **會彈出一個視窗**（360x480px）
   - 自動顯示當前網頁的標題和網址
   - 可以新增分類和標籤

4. **新增你的第一個分類**
   - 點擊分類旁邊的「+」按鈕
   - 輸入「測試分類」
   - 點擊「新增」

5. **新增標籤**
   - 點擊「+ 新增標籤」
   - 輸入「測試」
   - 按 Enter

6. **複製**
   - 點擊「📋 複製到剪貼簿」
   - 看到成功提示 ✓

7. **貼到 Notion**
   - 打開 Notion
   - 按 `Cmd+V` (Mac) 或 `Ctrl+V` (Windows)
   - 完成！

## 💡 快捷鍵（更快）

設定好後，可以用快捷鍵快速開啟：
- **Mac**：`Command + Shift + N`
- **Windows**：`Alt + Shift + N`

## 🔧 如果遇到問題

### 問題：點擊沒反應
**解決**：
1. 到 `chrome://extensions/`
2. 找到「WebPages Keeper」
3. 點擊重新整理圖示 🔄

### 問題：找不到擴充套件圖示
**解決**：
1. 點擊瀏覽器右上角的拼圖圖示 🧩
2. 找到「WebPages Keeper」
3. 點擊旁邊的「釘選」圖示 📌
4. 圖示就會固定在工具列
